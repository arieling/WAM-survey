## One-Sentence Summary

> F1 is a 4.2B Mixture-of-Transformer (MoT) framework with three specialized experts — Understanding (PaliGemma/SigLIP+Gemma), Generation (residual VQ-VAE, 10-scale next-scale prediction), and Action (flow-matching Gemma) — connected via hierarchical UGA progressive attention, achieving 95.7% LIBERO and 72.9% SimplerEnv while demonstrating that removing the generation expert causes a 12.6% drop on SimplerEnv.

---

## Core Contributions

1. **Mixture-of-Transformer (MoT) Architecture**: Three specialized expert modules — Understanding (perception), Generation (visual foresight), and Action (control) — with UGA progressive attention enforcing hierarchical information flow (understanding → generation → action) and preventing reverse shortcuts.
2. **Next-Scale Prediction for Visual Foresight**: Generation expert uses residual VQ-VAE decomposing frames into 16×16 patches across 10 spatial scales; autoregressive next-scale prediction produces multi-scale visual foresight as explicit planning targets for action generation.
3. **Three-Stage Training**: Stage I (foresight alignment: generation + frozen understanding), Stage II (full model pretraining on 330K trajectories), Stage III (task-specific post-training) — progressive training stabilizes multi-expert optimization.
4. **Action-Conditioned Foresight**: Action expert conditioned on visual foresight from generation expert — foresight provides explicit spatial planning context; action and foresight accuracy positively correlated despite modest (~40%) token accuracy.

---

## Problem Background

### Problem to Solve
Standard VLAs generate actions from observations directly without explicit future planning. Can a unified model that first generates visual foresight (predicted future states) and then generates actions conditioned on that foresight achieve better generalization and task success?

### Limitations of Existing Methods
- π₀: no visual foresight generation; 94.4% LIBERO, 48.3% SimplerEnv.
- gr00t-N1: no visual generation; 93.9% LIBERO, 30.4% real-world average.
- CoT-VLA: visual CoT but single-stage subgoal; not multi-scale.

### Motivation
Explicit visual foresight mirrors human planning — imagining the goal state before acting. The generation expert forces the model to predict future visual states, providing spatial grounding for action generation. UGA progressive attention ensures foresight is built on understanding and actions are built on foresight — not vice versa.

## Method Details

### Model Architecture

F1 is a **4.2B Mixture-of-Transformer** with three experts:

**Understanding Expert**:
- SigLIP vision encoder + Gemma decoder-only transformer
- PaliGemma architecture
- Inherits weights from π₀
- Encodes observations and language prompts

**Generation Expert**:
- Gemma backbone decoder (randomly initialized)
- Residual VQ-VAE: frames → 16×16 patches across k spatial scales (10 scales total, 4 at inference)
- Temporal convolutional network for motion feature aggregation
- Autoregressive next-scale prediction

**Action Expert**:
- Gemma backbone decoder
- Chunked action prediction with flow-matching objective
- Inherits from π₀ initialization
- Continuous action space

**Total**: 4.2B parameters; SigLIP vision encoder; Gemma with Swish, RMSNorm, Rotary Position Embeddings

### Core Modules

#### Module 1: UGA Progressive Attention

**Design Motivation**: Hierarchical causal flow from perception → foresight → control prevents the action expert from bypassing the foresight generation step (shortcutting) and ensures clean modular information flow.

**Implementation**:
- **Intra-expert**: Bidirectional attention within each expert
- **Inter-expert**: Hierarchical causal flow — understanding → generation → action
- **Generation expert**: Causal, scale-conditioned pattern for autoregressive consistency
- **No reverse flow**: Action expert cannot influence foresight generation

#### Module 2: Next-Scale Visual Foresight

**Design Motivation**: Multi-scale decomposition enables coarse-to-fine spatial prediction — early scales capture global structure (object positions), later scales add fine-grained details (textures, edges).

**Implementation**:
- Residual VQ-VAE: frames → 16×16 patches at 10 resolution scales

In Stage I, the generation expert is trained with teacher forcing on ground-truth visual tokens. The foresight alignment loss maximizes the likelihood of each ground-truth visual token $z_j$ given the preceding ground-truth sequence and the current observation history:

$$\mathcal{L}_{gen}^{gt} = -\mathbb{E}\left[\sum_j \log p_\theta(z_j \mid z_{1:j-1}^{gt}, \{o_i\}_{i=t-m}^t, l)\right]$$

In Stage II–III the model switches to predicting from its own autoregressive outputs ($\mathcal{L}_{gen}^{pred}$), and 4 scales are used at inference for speed.

#### Module 3: Flow-Matching Action Prediction

**Design Motivation**: Flow-matching provides a continuous-time interpolation objective for action prediction, avoiding the discretization artifacts of categorical cross-entropy while remaining computationally tractable.

The action expert is conditioned on visual foresight $q_t$ from the generation expert. Given a noisy action sample $a_t^\tau = (1-\tau)\epsilon + \tau a_t$ drawn from a linear interpolation between Gaussian noise $\epsilon$ and the target action $a_t$ at time $\tau \sim U(0,1)$, the model predicts the velocity needed to reach the clean action:

$$\mathcal{L}_{action} = \mathbb{E}\left[\|\pi_\theta(l, \{o_i\}_{i=t-m}^t, q_t, a_t^\tau) - (a_t - \epsilon)\|^2\right]$$

The visual foresight $q_t$ is a direct input to the action expert — conditioning action generation on the predicted future state.

#### Module 4: Three-Stage Training

**Stage II — Full Model Pretraining** and **Stage III — Task-Specific Post-training** both use a joint objective that balances foresight generation quality against action prediction accuracy:

$$\mathcal{L}_{total} = \mathcal{L}_{gen}^{pred} + \lambda \cdot \mathcal{L}_{action}$$

where $\lambda$ controls the relative weighting. The joint loss trains all three experts simultaneously, with the foresight loss encouraging accurate visual planning and the action loss enforcing execution quality.

**Stage II — Full Model Pretraining**:
- All experts trained jointly on 330K trajectories (136 tasks, 5 embodiments)

**Stage III — Task-Specific Post-training**:
- Fine-tuning on task-specific demonstrations
- Same objectives as Stage II

## Experiments

### Datasets

| Dataset | Scale | Characteristics | Usage |
|---------|-------|----------------|-------|
| LIBERO | Standard | 4 suites, tabletop | Fine-tuning/Evaluation |
| Open-X-Embodiment | Large | Multi-embodiment robot data | Pretraining |
| AgiBotWorld | Large | Diverse robot tasks | Pretraining |
| Total | ~330K trajectories, 136 tasks | 5 embodiments | Pretraining + fine-tuning |

### LIBERO Benchmark

F1 is evaluated across all four LIBERO suites, achieving the highest average success rate among the compared methods:

| Method | Spatial | Object | Goal | Long | Average |
|--------|---------|--------|------|------|---------|
| gr00t-N1 | 94.4% | 97.6% | 93.0% | 90.6% | 93.9% |
| π₀ | 98.0% | 96.8% | 94.4% | 88.4% | 94.4% |
| **F1 (pretrained)** | **98.2%** | **97.8%** | **95.4%** | **91.3%** | **95.7%** |

+1.3% over π₀ on LIBERO average; F1 best on Spatial (98.2%) and Goal (95.4%) — visual foresight helps goal-directed spatial tasks.

### SimplerEnv Bridge

SimplerEnv tests out-of-distribution generalization. F1's visual foresight provides the largest gains here, suggesting that imagined future states are most valuable when the environment distribution shifts:

| Method | SimplerEnv Average |
|--------|-------------------|
| π₀-Fast | 48.3% |
| SpatialVLA | 47.9% |
| F1 (not pretrained) | 66.1% |
| **F1 (pretrained)** | **72.9%** |

+24.6% over π₀-Fast on SimplerEnv; pretraining contributes +6.8%.

### Real-World Tasks (Genie Robot, 9 tasks, 15 trials each)

| Method | Average Success |
|--------|----------------|
| gr00t-N1 | 30.4% |
| π₀ | 65.2% |
| **F1** | **82.2%** |

+17% over π₀ on real-world tasks; large margin over gr00t-N1 (30.4%).

### Generation Expert Ablation

The ablation below demonstrates that the generation expert is essential for OOD generalization on SimplerEnv, while providing modest gains on the in-distribution LIBERO benchmark:

| Configuration | LIBERO Avg | SimplerEnv Avg |
|--------------|-----------|----------------|
| No Generation Expert | 94.4% | 60.3% |
| **Full F1** | **95.7%** | **72.9%** |

Generation expert provides +12.6% on SimplerEnv — visual foresight is critical for distribution shift generalization.

### Implementation Details

- **Model Size**: 4.2B parameters
- **Vision Encoder**: SigLIP (frozen)
- **Backbone**: Gemma (Swish, RMSNorm, Rotary Position Embeddings)
- **Image Tokenizer**: Residual VQ-VAE, 10 scales, 16×16 patches
- **Inference Scales**: 4 (reduced for real-time speed)
- **Historical Context**: m-step observation history
- **Action Space**: Continuous; flow-matching
- **Initialization**: Understanding + Action experts from π₀
- **Stage I**: Foresight alignment with teacher forcing
- **Stage II**: 330K trajectories, 136 tasks, 5 embodiments
- **Robot**: Genie (real-world), LIBERO sim

## Critical Analysis

### Strengths
1. MoT architecture provides clean modular separation while enabling joint training.
2. SimplerEnv (+24.6% over π₀-Fast) demonstrates strong OOD generalization from visual foresight.
3. Three-stage training effectively bootstraps multi-expert optimization.

### Limitations
1. **~40% image token accuracy**: Foresight quality is modest — suggests room for improvement in generation quality.
2. **4 scales at inference vs. 10 in training**: Reduction for speed may limit foresight quality.
3. **No code released**: Limited reproducibility.

### Potential Improvements
1. Higher-quality foresight (diffusion-based decoder) for better spatial grounding.
2. Full 10-scale inference with model distillation for speed.
3. Closed-loop foresight update mid-task for dynamic environments.
